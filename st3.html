css는 다음과 같은 3가지 방법으로 사용이 가능하다

인라인 - 사용 style HTML 요소 내부 속성

외부 - a를 사용하여 link 외부 CSS 파일에 연결할 요소


css 색상및 글꼴, 크기

CSS color 속성은 사용할 텍스트 색상을 정의한다.
CSS font-family 속성은 사용할 글꼴을 정의한다.
CSS font-size 속성은 사용할 텍스트 크기를 정의한다.

링크태그
target 속성은 연결된 문서를 열 위치를 지정한다.
그 target 속성은 다음 값 중 하나를 가질 수 있다.
_self - 기본값. 문서를 누른 것과 동일한 창/탭에서 열기
_blank - 새 창 또는 탭에서 문서 열기
_parent - 상위 프레임에서 문서 열기
_top - 창 전체 본문에서 문서 열기
예시 a href="https://www.w3schools.com/" target="_blank"> 보여지는 텍스트 /a
